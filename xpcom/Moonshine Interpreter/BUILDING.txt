WINDOWS (Rename the Makefile.win -> Makefile)

This software can built using GNU Make for Windows; 
which is part of the tools offered by MinGW:

http://www.mingw.com

UNIX/LINUX (Rename the Makefile.nix -> Makefile)

This software can be built using GNU Make, which will
probably be available as a part of development package for your 
distribution.

http://www.gnu.org/software/make/

BUILDING MOONSHINELUA COMPONENT

make (no arguments) - build the component
make generate - generate header and typelib
make clean - remove any files created during building component
make autoreg - copy .autoreg (this needs only be run once for any component)
make register - register component for Firefox
make remove - remove component from Firefox
